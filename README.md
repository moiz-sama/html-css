# HTML & CSS Projects

A collection of HTML and CSS projects showcasing various web design and layout techniques.

## Projects

### 1. Landing Page
A modern landing page design with responsive layout and styling.

### 2. Simple Blog Layout
A clean and simple blog layout with article sections and sidebar navigation.

### 3. Simple Survey Form
An interactive survey form with various input types and form elements.

### 4. eCommerce Page
An eCommerce product page with product listings and styling.

### 5. Sample Portfolio Page
A professional portfolio page to showcase projects and skills.

## Getting Started

Each project is self-contained in its own folder with:
- `index.html` - HTML structure
- `styles.css` - CSS styling

Simply open the `index.html` file in your web browser to view the project.

## Files Structure

## Notes

These are educational projects designed to practice HTML and CSS fundamentals including:
- Semantic HTML markup
- CSS flexbox and grid layouts
- Responsive design
- Form styling
- Color and typography

Feel free to modify and experiment with the code!
